Проект:"Инструменты Python для анализа открытых данных".
Цель проекта: 
Изучить и продемонстрировать применение инструментов языка программирования Python для анализа открытых данных.
Задачи:
- изучение основных библиотек Python для анализа данных;
- обзор технологий и инструментов Python для обработки открытых данных - анализ и сопоставление различных библиотек для анализа данных;
- разработка практических примеров с использованием инструментов Python для анализа данных;
- вывод по текущим тенденциям и перспективам использования Python в анализе данных.
